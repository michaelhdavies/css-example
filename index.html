This is my website for CSS styles!

Use this page as a quick reference for commonly used styling options.

Flexbox Row

Use to create a Flexbox row layout.

.row {
    display: flex;
    flex-direction: row;
    flex-wrap: wrap;
}

Flexbox Column

Use to create a Flexbox column layout.

.column {
    display: flex;
    flex-direction: column;
}

CSS Grid Layout

Use to build a 12-column CSS grid layout.

.grid {
    display: grid;
    width: 100%;
    grid-template-columns: repeat(12, 1fr);
}

Linear Gradients

Use to create a linear gradient from top to bottom.

.linear-gradient-background {
    background-image: linear-gradient(
        rgba(232, 102, 236, 0.3) 0%
        rgba(232, 102, 236, 0.3) 100%
    );
}

Box Transition Glow

Use transition and box shadows to glow on hover.

.code-card .card-header {
    border-radius: 8px;
    transition: all 0.5s ease-in-out;
}
.code-card:hover,
.code-card:hover .card-header {
    box-shadow: inset 0px 0px 8px 
        rgba(232, 102, 236, 1), 0 0 15px 
        rgba(232, 102, 236, 1);
}

Overlay Card with Title

Use position properties and negative margins to raise elements higher than natural starting point.

.card-header {
    position: relative;
    margin-top: -20px;
}
